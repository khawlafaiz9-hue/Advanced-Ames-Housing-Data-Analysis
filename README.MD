## 📌 نبذة عن المشروع | Overview
مشروع هندسة بيانات متقدم يعتمد على مجموعة بيانات **Ames Housing** الشهيرة لاستكشاف وتطهير وإعداد الميزات الجغرافية والهندسية للعقارات. يركز المشروع على معالجة البيانات الإحصائية وتجهيزها بشكل احترافي لتصبح صالحة وخالية من الانحياز لخوارزميات التنبؤ وتعلم الآلة.

---

## 🛠️ المراحل البرمجية وهندسة البيانات المتقدمة | Data Pipeline

يتضمن المشروع تطبيقاً عملياً لأعلى معايير معالجة البيانات (Data Preprocessing):

1. **Advanced Outlier Detection (رصد القيم المتطرفة):**
   * تحليل وتصفية البيانات برمجياً لعزل وتطهير القيم المتطرفة (Outliers) التي تؤثر سلباً على النمذجة الرياضية باستخدام العلاقات البيانية.

2. **Handling Missing & Skewed Data (معالجة البيانات المنحرفة والمفقودة):**
   * رصد واستبدال القيم المفقودة في الأعمدة الحجمية والفئوية بناءً على التوزيع الإحصائي السليم.
   * تطبيق تقنية **Log Transformation** لعلاج انحراف المتغيرات الرقمية (Skewed Features) وتحويلها إلى توزيع طبيعي (Normal Distribution).

3. **Feature Correlation Analysis (تحليل الارتباط الهندسي):**
   * بناء مصفوفة ارتباط كاملة (Correlation Matrix) ورسمها عبر خرائط حرارية (Heatmaps) لاستكشاف أقوى الميزات تأثيراً على المتغير المستهدف وسعر العقار النهائي.
